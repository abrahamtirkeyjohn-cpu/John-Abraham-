# Project Title

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/abrahamtirkeyjohn-cpu/John-Abraham-
   ```

2. Navigate to the project directory:
   ```bash
   cd John-Abraham-
   ```

3. Install the required dependencies:
   ```bash
   npm install
   ```

4. Start the application:
   ```bash
   npm start
   ```

5. Open your web browser and go to `http://localhost:3000` to view the project.

## Features List
- Interactive birthday greeting page
- Customizable greeting messages
- Ability to send WhatsApp greeting messages

## WhatsApp Phone Number Configuration Guide
To configure WhatsApp phone number for sending greetings:
1. Open the `.env` file in the root directory.
2. Add your WhatsApp phone number in the following format:
   ```
   WHATSAPP_NUMBER=+1234567890
   ```
3. Save the file and restart the application.

## Shareable Link Instructions
To generate a shareable link for the interactive birthday greeting page:
1. Navigate to the birthday greeting page in your browser.
2. Copy the URL from the address bar.
3. Share this link with friends and family to let them access the greeting page.
